## 📱 使用说明

### 1. 准备照片

把下面 3 张照片放进项目文件夹：

| 文件名 | 照片内容 |
|--------|----------|
| `photo-hiking.jpg` | 爬山时的合照 |
| `photo-concert.jpg` | 第一次演唱会照片 |
| `photo-skiing.jpg` | 滑雪照片 |

**照片建议：**
- 竖屏或正方形最佳
- 会自动模糊处理作为背景
- 选有氛围感的照片

### 2. 部署到 GitHub Pages

#### 方法一：网页上传（最简单）

1. 登录 GitHub，点击右上角 **+** → **New repository**
2. 仓库名填：`proposal-zhanghe-2025`
3. 选择 **Public**（免费）
4. 点击 **Create repository**
5. 上传文件：点击 **Add file** → **Upload files**
6. 把所有文件（index.html + 3张照片）拖进去
7. 点击 **Commit changes**

#### 方法二：命令行（如果你会用）

```bash
# 在 proposal-zhanghe-2025 文件夹内执行
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/proposal-zhanghe-2025.git
git push -u origin main
```

### 3. 开启 GitHub Pages

1. 在仓库页面点击 **Settings**
2. 左侧菜单找到 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 **main**，文件夹选 **/(root)**
5. 点击 **Save**
6. 等待 1-2 分钟，刷新页面，会看到绿色提示：
   > Your site is live at `https://xxx.github.io/proposal-zhanghe-2025/`

### 4. 手机上测试

- 复制上面的链接
- 发给微信文件传输助手
- 手机浏览器打开测试
